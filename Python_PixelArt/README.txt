Instrucciones:

1- El programa lee cada linea del archivo.
2- Si encuentra un Repetir, comprueba de que este sea de una sola linea o de más lineas. En cada caso, llama a la funcion Repetir.
3- Si no encuentra un Repetir, se separan las instrucciones en una lista, para ejecutarlas una por una.
4- Errores:
4.1- Solo se generará imagen si no existen errores en el programa.
4.2- Si el error es de escritura, pasa por las funciones de errores para integrarlas a una lista.
4.3- En esta lista no se repiten las lineas de error, ya que se comprueba que no exista en la lista.
4.4- Comprueba que esté bien escrita la segunda linea.
4.5- Comprueba que posterior del repetir y su entero, deba existir la palabra "veces".
4.6- Si existe un error que no pueda hacer funcionar el programa con normalidad, se activa el exit() y para el programa, estos errores pueden ser: 
    Que no exista un entero después de un repetir (ya que las intrucciones dentro de este repetir no servirán).
    Que al avanzar, la posicion esté fuera del margen de la matriz (Esto entregará las instrucciones exactas que efectuan el error).
    Que en la primera linea, no exista un numero después de la palabra Ancho
